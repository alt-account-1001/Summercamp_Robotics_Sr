# Connections for Obstacle avoiding

| pin from component     | pin on arduino         |
| ---                    | ---                    |
| **IR sensor**                                   |
| IR sensor VCC         | Arduino 5V pin          |
| IR sensor GND         | Arduino GND pin         |
| IR sensor DO          | Arduino digital pin 8   |
| **Motors**                                      |
| left  motor terminal 1 | Arduino digital pin 10 |
| left  motor terminal 2 | Arduino digital pin 11 |
| right motor terminal 1 | Arduino digital pin 12 |
| right motor terminal 2 | Arduino digital pin 13 |
